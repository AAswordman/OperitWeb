### How to Use Character Card (Prompt) Tags

![1. First, find Prompt Editing in Settings and click in, then find the Tags in the middle.](/manuals/assets/images/Luban_1756815566911eb5bf78d-36cd-408f-af27-560305fdab7b.jpg)
![2. You can choose to write your own tags or freely select configurations from the tag marketplace.](/manuals/assets/images/Luban_1756815566935c1a96719-c884-4a5f-8385-6a94af743497.jpg)
![3. Model marketplace. After entering, select what you need, click Add Tag, and you'll get this tag.](/manuals/assets/images/Luban_1756815566964c1471de7-4a2b-4423-b147-060c8f847a52.jpg)
![4. Ensure the tag is displayed successfully, return to "Character Cards"](/manuals/assets/images/Luban_1756821241174b27b752a-0a26-4597-9572-a898562f1a53.jpg)
![5. Select the character card you want to tag, click the pen in the upper right corner of the character card to enter edit mode](/manuals/assets/images/Luban_1756821241204212ddc6b-dec2-479a-9aeb-7a79a2faa9e7.jpg)
![6. In the character card editing interface, activate the "Tag" you just created, and finally click "Save" to complete the tag setup.](/manuals/assets/images/Luban_17568212412371594017f-fc6c-4245-b900-e275ef85c078.jpg)
